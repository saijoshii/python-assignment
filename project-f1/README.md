F1 Lap Timing Processor
Overview

This Python program processes F1 lap timing data from a text file and displays various statistics, including:
    Race name
    Fastest driver and their time
    Driver statistics (fastest lap, average lap, and lap count)
    Leaderboard in descending order of fastest lap times

After running the program, in terminal run this command python main.py and txt file name and enter.
 for example: python main.py lap_times_1.txt 

 If this is shown "Error: No such file or directory" then
 type cd project-f1 and enter
 i.e    cd project-f1
