 University of Poppleton Chatbot.

 Overview
 This is the chat bot of Poppleton uniersity for answering quesyion related to it.It can help users with details like coffee shop hours, parking, library timings, parking, and more. Admins can manage Q&A pairs and view user logs.

Features
 User Interaction:
    1.Answer common university questions.
    2.If the keywords doesnot match it provides random responses.
    3.Log user interactions.
    4.Random disconnections.

Admin Mode:
    1.Add and manage question and answers.
    2.View existing question/answer and user logs.

Commands
User Commands:
    Type any question related to the University of Poppleton (for example: "What time does the coffee shop open? ").
    Type help to display the the help desk.
    Type bye, quit, exit, goodbye, later to end the conversation.

Admin Commands:
    Type admin to enter the admin page.

Files Created:
    1.qa_data.json to store question-answer.
    2.user_log.txt logs user queries and corresponding responses for administrative review.
